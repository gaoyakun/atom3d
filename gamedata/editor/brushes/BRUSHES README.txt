To add new Brushes to Earthsculptor 

They must be
1. grayscale 8 bit per pixel/channel
2. png files
3. pixel height and width must be identical and must be power of 2 (ex: 64,128,256...)
4. in the current brushes folder
5. brush should not extend to the edges as it will get cut off


The default circle brush cannot be changed, but you can modify/replace the other brushes
The brushes have are loaded in order by their location number in the filename
The unregistered version has a limit of 5 brushes total
The registered version has a limit of 30
